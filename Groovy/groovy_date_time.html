Working with Date and Time

In integration scenarios, you often need to handle dates and times for logging, filtering, or data mapping. Groovy's date and time capabilities simplify these tasks significantly.

Getting the Current Date and Time

You can easily get the current date and time and format it for your needs.

import java.text.SimpleDateFormat

def now = new Date();
def formatter = new SimpleDateFormat("yyyy-MM-dd'T'HH:mm:ss");
def formattedDate = formatter.format(now);
// formattedDate will be something like "2024-05-20T10:30:00"
message.setHeader("currentTimestamp", formattedDate);

Parsing a Date String

When you receive a date in a specific format, you can parse it into a `Date` object to perform calculations or reformat it.

import java.text.SimpleDateFormat

def dateString = "2024-05-15";
def formatter = new SimpleDateFormat("yyyy-MM-dd");
def parsedDate = formatter.parse(dateString);
// Add a few days to the date
def futureDate = parsedDate + 5;
message.setHeader("futureDate", futureDate.toString());
