1. Introduction to SAP CPI Fundamentals

In this section, you will learn the foundational concepts of SAP Cloud Platform Integration (CPI) within the broader SAP ecosystem. We will cover the key terminology and the basic architecture that makes CPI a powerful and flexible integration platform.

What is SAP Integration Suite?

SAP Integration Suite is a comprehensive, hybrid, and open integration platform-as-a-service (iPaaS) that simplifies and accelerates your integration journey. It provides a wide range of tools and services to connect SAP and non-SAP applications in various environments, including cloud-to-cloud and cloud-to-on-premise scenarios.

The Role of SAP CPI

Within the Integration Suite, SAP CPI (Cloud Platform Integration) is the core component for building integration flows (iFlows). It is a highly versatile and secure solution for designing, developing, and deploying integrations. It allows you to transform data, apply business logic, and route messages between different systems without the need for complex, point-to-point connections.

Basic iFlow Design Concepts

An iFlow represents a complete integration scenario. It consists of a series of processing steps that define how a message is received, transformed, and sent to its destination. This module will introduce you to the fundamental components of an iFlow, such as:

  • **Sender Channel:** The entry point for a message.
  • **Receiver Channel:** The exit point where the message is delivered.
  • **Message Mappings:** Used for converting data structures.
  • **Content Modifier:** Allows you to add, modify, or delete message headers, body, and properties.
